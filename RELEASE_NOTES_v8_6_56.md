# SmartAdvisor v8.6.56-polished-final

Финальная полировка после production-fixed аудита. Цель релиза — убрать последние UX-риски, из-за которых пользователь мог получить слишком хороший результат без реальной самостоятельной сборки решения.

## Что изменено

1. **Честный визуальный сборщик учебных решений**
   - Контроли больше не выбраны по умолчанию.
   - Слабый черновик, пользовательский выбор и эталон разделены.
   - Пользовательский payload строится серверно только из реально выбранных контролей.
   - Если контроли не выбраны, эталонные элементы не протекают в решение.

2. **Новый API для visual builder**
   - Добавлен endpoint `/api/learning/visual-payload`.
   - Поддержаны режимы `weak`, `selected`, `reference`.
   - Для `selected` учитывается список выбранных controls.

3. **Убрана гонка в UI**
   - Проверка решения теперь дожидается сборки payload.
   - Кнопки визуального конструктора работают предсказуемо.

4. **Мобильная полировка**
   - Улучшены размеры чекбоксов, заголовков и action-зоны.
   - Desktop/tablet/mobile проверены на отсутствие горизонтального overflow.

5. **Регрессионные тесты**
   - Добавлены проверки, что пустой пользовательский выбор не равен эталону.
   - Проверено, что частичный выбор не подтягивает невыбранные production-контроли.
   - Проверено, что reference-режим по-прежнему подставляет эталон.
