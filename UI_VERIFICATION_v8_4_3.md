# Проверка v8.4.3 wizard UI

Дата проверки: 2026-06-14

## Что изменено

- Основной вход изменён с большого экрана выбора на пошаговый wizard: пользователь видит один вопрос за раз.
- Старые примеры оставлены в свернутом блоке и не являются основным путём.
- В простом режиме карточки систем и шагов стали читаемыми: без кнопок `+ до`, стрелок, копирования и технических полей.
- Техническое редактирование осталось в экспертном режиме.
- Усложнения сгруппированы по человеческим причинам: надёжность, потоки и системы, данные и аналитика, бизнес-риск.
- Верхнее меню упрощено: `Собрать процесс`, `Проверки`, `База знаний`.
